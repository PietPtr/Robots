Basics

Click on parts to select them. Once selected they turn green. Click 'Add to code' to insert an expression into the code at the current cursor position that evaluates to the selected component. Then just use the functions listed below for the relevant parts to make the robot move.

The delta variable is the amount of seconds since the last frame.

Servo

.rotate(radials)

Rotate servo's as follows: robot.part("abcdef01").rotate(delta); This will rotate the servo at 2 PI radials per second.

.angle()

Get the current angle of the servo.
